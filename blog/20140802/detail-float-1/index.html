Float详解（一）

2014 年8 月2 日 12:00 am | Comments

一、float的基本原理

float属性定义元素相对于一个块级元素左、右方向的浮动。当前float元素是处于流动状态（flow statu），实际还是脱离了普通流（normal flow）。float是处于流动状态，与absolute和relative的脱离普通流，处于另外的层级有一定区别。

其实说起来抽象了一点，最傻瓜化的理解我认为就从float本身的名字去理解，即：浮动。既然说到浮动，有一个形象的实物可以对应，那就是“水”。

“水”有一个特性，就是流动->浮动。

打个比方，一个岛屿，其中一片水是一个层级，某一个元素浮动过后，就想像成水上的漂浮物。这样子相对于陆地上固定的物体来说，漂浮物是占据了岛屿空间的。

（1）float元素的宽高：如果那片水没有长和宽的属性来约束，那么漂浮物的空间其实就可以理解为水的空间。

（2）float的值：如果有两块木板都是漂浮物，设置不同的漂浮方向，肯定会看上去不一样。

（3）float的行：如果水的区域只有：10m * 10m，我们现在放置20个木板，每个木板规格是1m * 1m，那么，将漂浮成为两排。

对于float元素的宽、高，下面来看一个实际的例子：

<div class="wrap">这是一个漂浮物。</div>

.wrap {
	float: left;
	color: #fff;
	background: #acc;
}

tip：当块级元素未设置宽、高的时候，如果是float元素，那么宽高就根据内容来自适应。如果没有高度即元素不可见。

其实float有一个“行”的概念，对于“行”的理解，我们可以理解为：一个块级元素所在的水平空间。形象的比喻就是一个有高度的块级元素，在浏览器中所在的位置，默认平铺为一行。

二、float之图文混排

float最开始我觉得就是来做图文混排的，使用float来做文字围绕图片的效果实在是太简单不过了。

<div class="wrap">
	<img class="red-img" src="img/red.png" alt="红色图">
	<p class="aritcle">这里是一个红色背景的图片，现在如果文字过多，设置为浮动，会出现围绕图片的经典布局。</p>
</div>

<button class="btn－star">开始</button>
<button class="btn-stop">停止</button>

body {
	margin: 0;
	padding: 0;
	background: #eee;
}
.wrap {
	width: 150px;
	height: 200px;
	background: #fca;
}
.wrap .red-img {
	float: left;
}

如果要实现文字不围绕图片的效果，使用闭合浮动的方式即可，对于闭合浮动的值，单独开一篇解释。

.wrap .aritcle {
	clear: left;
}

三、float之高度塌陷

当内部元素设置float后，外部元素会出现高度塌陷的情况，具体看如下实例：

<div class="wrap">
	<div class="inner-wrap">这里是内容区域</div>
</div>			

.wrap {
	width: 600px;
	height: 200px;
	background: #fcc;
}
.wrap .inner-wrap {
	width: 400px;
	height: 220px;
	float: left;
	background: #acc;
}

下面举一个方便理解的例子，就是平时用无序列表的时候会遇到ul标签的高度在li标签float过后塌陷的问题，解决方案就是加一点css即可（当然网上有很多方案，好像有6，7种，可以查阅）。

<ul class="items">
	<li class="item html">HTML</li>
	<li class="item css">CSS</li>
	<li class="item js">JavaScript</li>
</ul>

ul {
	margin: 0;
	padding: 0;
	list-style: none;
}
.items .item {
	float: left;
	width: 100px;
	height: 30px;
	margin-left: 10px;
	display: inline;
}
.items .html {
	background: #ace;
}
.items .css {
	background: #acc;
}
.items .js {
	background: #fcc;
}

解决方案：

<ul class="items clearfix">...</ul>

.clearfix:after {    
	display: block;
	content: "";
	height: 0;
	line-height: 0;
	clear: both;    
	visibility: hidden;
}
.clearfix {
	zoom: 1; /* ie6 */
}

tip：其实相当于在父级元素末尾加一个高度为0的空元素，在这个空元素的基础上使用clear: both（后面一篇解释）。
