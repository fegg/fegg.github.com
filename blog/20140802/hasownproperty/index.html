hasOwnProperty

2014 年8 月2 日 12:00 am

英文版本：https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Object/hasOwnProperty

一、概要

hasOwnProperty()方法返回一个布尔值，这个布尔值表示：当前对象是否包含指定的属性。

二、语法

obj.hasOwnProperty(prop);

参数介绍

prop: 属性的名字，类型是string

三、描述

每一个对象都会从Object继承hasOwnProperty()方法。这个方法可以用来确定当前对象的指定对象是否是直接对象。和”in”操作符不同的是，hasOwnProperty()方法不检查当前对象原形链中的属性。

  1. 一个对象自己的属性指的是：直接对该对象定义的属性。并非是该对象的原形继承的属性。
  2. 对象的属性包括：变量、函数。

四、例子

（1）使用hasOwnProperty()来判断一个属性是否存在。

以下例子就是确定 o 这个对象是否包含一个名为prop的属性：

var o = new Object();
o.prop = 'exits';

function change (o) {
    o.newProp = o.prop;
    //删除属性
    delete o.prop;
}

var ret1 = o.hasOwnProperty('prop');
change(o);
var ret2 = o.hasOwnProperty('prop');

console.log('ret1: ' + ret1 + ', ret2: ' + ret2);

（2）指向直接属性和继承属性

var o = new Object();
o.prop = 'exists';
//直接属性
o.hasOwnProperty('prop');
//继承属性，注意没有加括号，如果加入了括号就是调用函数
o.hasOwnProperty('toString');
o.hasOwnProperty('hasOwnProperty');
