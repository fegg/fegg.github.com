JQuery选择器优化策略

2014 年8 月2 日 12:00 am | Comments

1. 多用ID选择器

1) getElementById：是浏览器的内置函数，所以效率比较高；

2) getElementsByTag：也是浏览器内置函数，效率由于唯一性，低于id选择器；

3) getElementsByClassName：浏览器没有内置函数，我们要解析HTML文档树来外递归实现选择，所以效率很慢。

对于选择器，w3c官方有全面介绍，其实JQuery的选择器模块参考的CSS，这里给出浏览器对CSS 2.1和CSS 3的支持情况。

CSS 2.1

CSS 3

2. 少直接使用class选择器，可以适当使用符合选择器

可以过滤结果集，这样可以提升一下选择的性能。JQuery的过滤部分做得还是很优秀的。

tag.class

3. 多用父子关系，少用嵌套关系

//父子关系
parent>child
//嵌套关系
outer inner

4. 缓存JQuery对象

//先缓存，再循环操作，这里将函数调用也避免了循环调用
var dom = $('#id'), content = '';
for(var i = 0; i < length; i++) {
    content += 'something';
}
dom.append(content);

5. 使用链式操作，避免重复选择对象

var dom = $('#id').css('color', '#fcc').append('<div>something</div>');
